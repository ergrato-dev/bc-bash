> **Tiempo estimado**: 4-6 horas | **Nivel**: Intermedio

## 🎯 Objetivos de la Semana

Al completar esta semana serás capaz de:

- ✅ Declarar y usar variables en Bash
- ✅ Aplicar expansión de parámetros
- ✅ Usar condicionales `if/elif/else` y `case`
- ✅ Implementar bucles `for`, `while` y `until`
- ✅ Crear scripts interactivos con menús
- ✅ Validar entrada del usuario

## 📋 Requisitos Previos

- Completar [Semana 2](../week-02-pipes-redireccion-y-procesamiento-de-texto/README.md)
- Entender comandos básicos de terminal
- Saber crear y ejecutar scripts básicos
