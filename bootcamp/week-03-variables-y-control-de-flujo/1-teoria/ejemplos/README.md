# Ejemplos de Variables y Control Flow - Semana 3

Este directorio contiene ejemplos prácticos para la semana 3 del Bootcamp Bash Scripting.

## 📁 Estructura

```
week-03-variables-y-control-de-flujo/
├── ejemplos/           # Scripts de ejemplo para aprendizaje
├── ejercicios/         # Ejercicios prácticos
├── proyectos/          # Proyectos de la semana
└── datos/              # Archivos de datos para práctica
```

## 🎯 Contenido de la Semana

### Variables y Expansión de Parámetros

- Variables de entorno
- Expansión de parámetros `${variable}`
- Valores por defecto `${var:-default}`
- Longitud de variables `${#var}`
- Transformaciones `${var^^}`, `${var,,}`

### Estructuras de Control

- Condicionales `if/else/elif`
- Switch con `case`
- Operadores de comparación
- Pruebas de archivos

### Bucles

- Bucle `for` con rangos y listas
- Bucle `while` con condiciones
- Bucle `until` con condiciones inversas
- Control de flujo con `break` y `continue`

### Argumentos y Opciones

- Variables especiales `$0`, `$#`, `$*`, `$@`
- Procesamiento con `getopts`
- Validación de argumentos
- Scripts interactivos

## 🛠️ Uso

Los scripts están organizados por dificultad y tema. Cada ejemplo incluye:

- Comentarios explicativos
- Casos de uso prácticos
- Manejo de errores
- Buenas prácticas
