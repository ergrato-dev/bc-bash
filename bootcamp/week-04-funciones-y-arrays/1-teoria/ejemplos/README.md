# Ejemplos de Functions & Arrays - Semana 4

Este directorio contiene ejemplos prácticos para la semana 4 del Bootcamp Bash Scripting.

## 📁 Estructura

```bash
week-04-funciones-y-arrays/
├── ejemplos/           # Scripts de ejemplo para aprendizaje
├── ejercicios/         # Ejercicios prácticos
├── proyectos/          # Proyectos de la semana
└── datos/              # Archivos de datos para práctica
```

## 🎯 Contenido de la Semana

### Funciones en Bash

- Definición y sintaxis de funciones
- Parámetros y argumentos
- Variables locales vs globales
- Valor de retorno y códigos de salida
- Funciones recursivas
- Bibliotecas de funciones reutilizables

### Arrays y Estructuras de Datos

- Arrays indexados básicos
- Arrays asociativos (diccionarios)
- Operaciones con arrays
- Iteración y manipulación
- Arrays como parámetros de funciones
- Estructuras de datos complejas

### Combinaciones Avanzadas

- Funciones que operan sobre arrays
- Bibliotecas de utilidades
- Patrones de diseño en Bash
- Modularización de código
- Reutilización y mantenibilidad

## 🛠️ Uso

Los scripts están organizados por complejidad y tema. Cada ejemplo incluye:

- Documentación clara y comentarios
- Casos de uso prácticos
- Manejo de errores robusto
- Ejemplos de testing
- Buenas prácticas de programación

## 📚 Scripts Disponibles

- `funciones_basicas.sh` - Fundamentos de funciones
- `funciones_avanzadas.sh` - Recursión y casos complejos
- `arrays_indexados.sh` - Arrays básicos y operaciones
- `arrays_asociativos.sh` - Diccionarios y mapas
- `funciones_arrays.sh` - Combinación de conceptos
- `biblioteca_utilidades.sh` - Funciones reutilizables
