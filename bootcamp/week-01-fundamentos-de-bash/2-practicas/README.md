# 🏋️ Prácticas - Semana 01

## 📋 Descripción

Ejercicios guiados para practicar los conceptos fundamentales de la terminal y Bash.

## 🎯 Metodología

Cada práctica sigue el formato **aprender descomentando**:

1. Lee el `README.md` de la práctica
2. Abre `starter/script.sh`
3. Descomenta el código según las instrucciones
4. Ejecuta y verifica los resultados

## 📚 Lista de Prácticas

| #   | Práctica                              | Descripción                      | Duración |
| --- | ------------------------------------- | -------------------------------- | -------- |
| 01  | [Navegación](practica-01-navegacion/) | Explorar el sistema de archivos  | 30 min   |
| 02  | [Permisos](practica-02-permisos/)     | Gestionar permisos de archivos   | 30 min   |
| 03  | [Comandos](practica-03-comandos/)     | Manipular archivos y directorios | 45 min   |

## ⏱️ Tiempo Total Estimado

**~2 horas** de práctica guiada

## 📝 Instrucciones Generales

### Preparación

```bash
# Navegar a la carpeta de prácticas
cd bootcamp/week-01-fundamentos-de-bash/2-practicas/

# Dar permisos de ejecución a los scripts
chmod +x practica-*/starter/*.sh
```

### Ejecución

```bash
# Ejecutar script starter
./practica-01-navegacion/starter/script.sh
```
